##Descrição
Este projeto consiste na implementação de um sistema de fila de impressão utilizando o conceito de FIFO (First In, First Out), onde os documentos são processados na ordem em que chegam.
O sistema também inclui um mecanismo de prioridade, permitindo que documentos urgentes sejam impressos antes dos demais.

 ##Funcionalidades
- Adicionar documento à fila
- Imprimir documento (remoção da fila)
- Mostrar fila de impressão
- Prioridade de impressão (documentos urgentes têm preferência)

 ##Linguagem Utilizada
Linguagem C
