# Exercício 3 – Fila usando vetor

Uma fila também pode ser implementada usando um vetor (array).

Nesse caso, utilizamos duas posições principais:
- Início → onde os elementos são removidos
- Fim → onde novos elementos são adicionados

Quando um elemento é inserido, ele vai para o fim da fila. Quando é removido, sai do início.
No entanto, essa abordagem tem uma limitação: o vetor possui tamanho fixo, o que pode restringir o crescimento da fila.
