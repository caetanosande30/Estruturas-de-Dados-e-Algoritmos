# Exercício 5 – O que é escalonamento de processos

Escalonamento é o processo utilizado pelo sistema operacional para decidir qual processo será executado primeiro.

É como um gestor que organiza a ordem de execução das tarefas.

Existem diferentes tipos de escalonamento:
- FIFO → segue a ordem de chegada
- Round Robin → cada processo recebe um tempo de execução
- Prioridade → processos mais importantes são executados primeiro

No projeto desenvolvido, foi utilizado o conceito de prioridade.
