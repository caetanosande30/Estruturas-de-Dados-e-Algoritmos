# Exercício 4 – Uso de filas no sistema operacional

Os sistemas operacionais utilizam filas para organizar a execução de processos.

Por exemplo, quando vários programas estão abertos ao mesmo tempo, todos precisam usar o processador. O sistema organiza esses processos em uma fila para garantir que cada um seja executado na sua vez.

Isso permite que o sistema funcione de forma organizada e eficiente.
