# Exercício 1 – O que é uma fila (FIFO)?

Uma fila é uma forma de organizar elementos onde o primeiro que entra é o primeiro a sair. Esse conceito é conhecido como FIFO (First In, First Out).

Um exemplo simples do dia a dia é uma fila no banco ou no supermercado. A pessoa que chega primeiro é atendida primeiro, enquanto as outras seguem a ordem de chegada.

Isso garante organização e justiça no atendimento.
